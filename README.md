# Repetition-UBER
Snabb prototyp kopa av Uber.com/se/en


## Förbättringar
- Såklart så är sidan inte responsiv för fem öre vilket är något man lätt skulle kunna lägga till om man bara lägger ner nån timma på det. 
- Font storlekar samt paddings o margins behövs fixas på vissa ställen. 
- Valet av HTML taggar kan förbättras rätt mycket, samt namnen på CSS klasser/id's
- CSS kunde skrivas lite mer effektivare också

## Almänna Tankar
Egentligen skulle jag gjort hemsidan för mobiler först om man tänker ur ett responsivt perspektiv, då det är mycket lättare (enligt mig) att "scale:a" upp det till större skärmare än att "scale:a" ner till mindre skärmar. Sen tycker jag inte heller att sidan är så jätte bra uppbyggd men tyckte det räckte för denna initiella uppgiften.
